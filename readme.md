# **musiCortex (Before Zunefy)** - *A Spotify Web API CRUD with a Microsoft Zune UI inspiration.*

## Setup
```
    npm install
```
Don't forget the .env file.
```
    PORT=""
    MONGODB_URI=""
    SECRET=""
    CLIENT_ID="" //From spotify web api 
    CLIENT_SECRET="" //From spotify web api
```
